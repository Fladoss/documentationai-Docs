---
title: Cannot Upload File
description: Fix problems when uploading files to Cuddly Records.
---

## Problem

- File upload fails.
- Error: "Upload unsuccessful."

## Possible Causes

- File size too large.
- Unsupported file type.

## Quick Fix

<Steps>
  <Step title="Check file size">
    Ensure your file is under 100MB.
  </Step>
  <Step title="Check file format">
    Accepted types: MP3, FLAC, WAV, JPG, PNG.
  </Step>
  <Step title="Try another browser">
    If issues persist, switch browsers and retry.
  </Step>
</Steps>
