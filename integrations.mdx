## Setting Up Webhooks

Webhooks notify external services of events like doc updates or publishes. Follow these steps to configure one.

<Steps>
  <Step title="Create Webhook" icon="plus">
    Navigate to https://dashboard.example.com/settings/webhooks and click "New Webhook".
  </Step>
  <Step title="Configure Events" icon="settings">
    Select events such as `doc.updated` or `doc.published`.
  </Step>
  <Step title="Add Endpoint" icon="link">
    Enter your endpoint URL, like `https://your-webhook-url.com/cuddly-records`.
  </Step>
  <Step title="Test and Save" icon="check-circle">
    Send a test payload and save.
  </Step>
</Steps>

Handle incoming webhooks with this example:

<CodeGroup tabs="Node.js,Python">
  ```javascript
  const express = require('express');
  const app = express();
  app.use(express.json());

  app.post('/cuddly-records', (req, res) => {
    const event = req.headers['x-cuddly-event'];
    console.log(`Event: ${event}`, req.body);
    res.status(200).send('OK');
  });

  app.listen(3000);
  ```
  ```python
  from flask import Flask, request, jsonify
  app = Flask(__name__)

  @app.route('/cuddly-records', methods=['POST'])
  def webhook():
      event = request.headers.get('X-Cuddly-Event')
      print(f"Event: {event}", request.json)
      return jsonify({'status': 'OK'}), 200

  if __name__ == '__main__':
      app.run(port=3000)
  ```
</CodeGroup>

## Exporting and Importing Docs

Export your documentation in various formats or import from external sources.

<Tabs>
  <Tab title="Export to Markdown" icon="file-text">
    Use the API to fetch docs:
    
    <Request tabs="cURL,JavaScript">
      ```bash
      curl -H "Authorization: Bearer YOUR_API_KEY" \
           https://api.example.com/v1/docs/export/markdown?doc_id=123
      ```
      ```javascript
      const response = await fetch('https://api.example.com/v1/docs/export/markdown?doc_id=123', {
        headers: { 'Authorization': 'Bearer YOUR_API_KEY' }
      });
      const markdown = await response.text();
      ```
    </Request>
  </Tab>
  <Tab title="Import from JSON" icon="upload">
    <ParamField path="docs" param-type="array" required="true">
      Array of doc objects with `title` and `content`.
    </ParamField>
    
    <Request show-lines="false">
      ```bash
      curl -X POST -H "Authorization: Bearer YOUR_API_KEY" \
           -H "Content-Type: application/json" \
           -d '{"docs": [{"title": "New Doc", "content": "# Hello"}]}' \
           https://api.example.com/v1/docs/import/json
      ```
    </Request>
  </Tab>
</Tabs>

## API Access for Custom Integrations

Build bespoke integrations using the REST API. Authenticate with `YOUR_API_KEY` from your dashboard.

<Response>
  ```json
  {
    "docs": [
      {
        "id": "doc_123",
        "title": "API Guide",
        "updated_at": "2024-10-15T10:00:00Z"
      }
    ]
  }
  ```
</Response>

<Callout kind="tip">
  Generate a new API key at https://dashboard.example.com/api-keys. Rotate keys regularly for security.
</Callout>

<Expandable title="Advanced: Webhook Signature Verification">
  Verify payloads using HMAC SHA-256 with your secret:
  
  ```javascript
  const crypto = require('crypto');
  const signature = 'sha256=' + crypto.createHmac('sha256', 'YOUR_WEBHOOK_SECRET')
    .update(payload)
    .digest('hex');
  ```
</Expandable>
