## Overview

Cuddly Records provides a centralized hub for your project documentation. You can organize pages, embed code snippets, and collaborate seamlessly. This space scales with your needs, from simple READMEs to comprehensive guides.

## Key Features

Cuddly Records offers powerful tools to streamline your documentation workflow.

## Quick Start

Get up and running in minutes with these steps.

<Steps>
  <Step title="Create Account" icon="user-plus">
    Sign up at `https://dashboard.example.com/register` using your email.
  </Step>
  <Step title="Set Up Space" icon="plus">
    From the dashboard, click "New Space" and name it "My Project Docs".
  </Step>
  <Step title="Add First Page" icon="file-plus">
    Navigate to your space and create a page titled "README". Add content using the editor.
  </Step>
  <Step title="Publish" icon="globe">
    Click "Publish" to make your docs live and share the URL with your team.
  </Step>
</Steps>

## Embed Examples in Your Docs

Showcase code directly in pages for better clarity.

<CodeGroup tabs="JavaScript,Python">
  ```javascript
  // Fetch your docs
  const response = await fetch('https://api.example.com/docs/YOUR_SPACE_ID', {
    headers: { Authorization: `Bearer ${YOUR_API_KEY}` }
  });
  const docs = await response.json();
  console.log(docs);
  ```
  ```python
  # Fetch your docs
  import requests
  response = requests.get(
      'https://api.example.com/docs/YOUR_SPACE_ID',
      headers={'Authorization': f'Bearer {YOUR_API_KEY}'}
  )
  docs = response.json()
  print(docs)
  ```
</CodeGroup>
